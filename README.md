# Low-Power FIFO-Based Router

This project implements a Low-Power FIFO-Based Router using Verilog HDL. The router uses FIFO buffers to temporarily store incoming data and route it to the correct output path based on control logic. The design focuses on digital logic design, data buffering, routing operation, and basic low-power design concepts.

## Features
- FIFO-based data buffering
- Verilog HDL implementation
- Router control logic
- Synchronous design
- Low-power design approach
- Testbench for functional verification

## Tools Used
- Verilog HDL
- Xilinx Vivado / ModelSim
- GitHub

## Applications
- Digital communication systems
- Network-on-Chip basics
- VLSI/RTL design practice
- Data packet routing systems
